## 🚀 快速开始

<details open>
<summary><b>📦 环境要求</b></summary>
<br>

- [Node.js](https://nodejs.org/) `>= 12.x`
- [Git](https://git-scm.com/) `最新版`

</details>

<details open>
<summary><b>🔧 本地运行</b></summary>
<br>

```bash
# 克隆仓库
git clone https://github.com/New-luo/New-luo.github.io.git
cd New-luo.github.io

# 安装依赖
npm install

# 启动开发服务器 🚀
npm run server
```
> 浏览器访问 **http://localhost:4000** 即可预览

</details>

<details>
<summary><b>📦 常用命令</b></summary>
<br>

```bash
npm run clean      # 🧹 清理缓存
npm run build      # 🏗️ 生成静态文件
npm run server     # 🚀 启动本地服务器
npm run deploy     # 📤 部署到 GitHub Pages

# 三件套一键启动
hexo clean && hexo g && hexo s
```

</details>

## 📁 目录结构

```
📦 anzhiyu-blog
├── 📋 _config.yml              ← Hexo 主配置
├── 🎨 _config.anzhiyu.yml      ← 主题配置（重要）
```

## 📝 写文章

在 `source/_posts/` 下创建 `.md` 文件，或使用命令快速创建：

```bash
hexo new "我的新文章"
```

**Front Matter 模板：**

```yaml
---
title: 文章标题
top_img:   https://example.com/top.jpg      # 顶部大图
cover:     https://example.com/cover.jpg     # 列表封面
categories: [Hexo, 教程]
tags:       [标签1, 标签2, 标签3]
---
```
